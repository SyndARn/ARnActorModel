ARnActor Actor Model solution for .NET C

ARnActor is a small Actor Model Library for C#.

This is a first draft, so be patient for anything else.

To create your first actor :

var actor = new actActor() ;

To send a message to another actor : var receiver = new actActor() ;

actor.SendMessageTo("Hello world",receiver) ;

Some samples are furnished around a small console application, try :

  • disco => give you the well known public actors

  • Ring => a performance test (12 s expected)

    There is no required Library or anything else : what you need is a CLR NET 4.5, also it could run on CLR NET 4.0.
